GOLDBECK OPTIMIZER - CODE AUDIT QUICK REFERENCE
================================================

CRITICAL ISSUES (Fix before deployment)
======================================

1. RACE CONDITION: Job state (lines 21, 31-124 in floorplan.py)
   - Background thread writes to _jobs[job_id] while HTTP thread reads
   - FIX: Add threading.Lock() around all _jobs accesses

2. EVENT LOOP BUG: Missing asyncio.set_event_loop() (lines 44-49 floorplan.py)
   - WebSocket broadcasts fail silently to Grasshopper
   - FIX: After asyncio.new_event_loop(), call set_event_loop(loop)

3. DEPRECATED DATETIME: datetime.utcnow() (lines 64, 82 in ifc_exporter.py)
   - Breaks on Python 3.12+
   - FIX: Replace with datetime.now(timezone.utc)

4. HARDCODED US COORDS: Denver default (lines 25, 37 in plot.py)
   - German app defaults to USA location
   - FIX: Fail explicitly or use German default

5. MEMORY LEAK: Unbounded job cache (lines 21, 18 in floorplan.py, optimize.py)
   - _jobs and jobs dicts grow forever → OOM crash
   - FIX: Implement TTL cleanup, max 2000 jobs

6. ARRAY BOUNDS: Unsafe variants[0] (lines 92-93 in floorplan.py)
   - Crashes if optimizer returns empty list
   - FIX: Check len(variants) > 0 before access

HIGH-SEVERITY ISSUES (Fix this week)
====================================

7. BROAD EXCEPTION HANDLING: except Exception: pass (30+ occurrences)
   - All errors silently swallowed
   - FIX: Add logger.error(..., exc_info=True) to every catch

8. WS BROADCAST FAILURES: Silent fire-and-forget (lines 52-59, 106-107 floorplan.py)
   - Grasshopper plugin never gets progress
   - FIX: Log all broadcast errors

9. GLOBAL HTTP CLIENT: No thread safety (lines 82-93 cadastral.py)
   - Multiple clients created simultaneously
   - FIX: Use asyncio.Lock() or context manager

10. JSON PARSE ERRORS: Silent failure (lines 78-92 parcel_store.py)
    - Corrupted parcels show zero area
    - FIX: Log parse errors, set defaults carefully

11. NO RATE LIMITING: WFS endpoints exposed (cadastral.py, parcels.py)
    - Can DDoS external services
    - FIX: Add slowapi limiter, 10/min per IP

12. NO TRANSACTION ISOLATION: SQLite default (database/engine.py)
    - Concurrent writes cause data corruption
    - FIX: Use session.begin() for explicit transactions

MEDIUM-SEVERITY ISSUES (Fix this month)
=======================================

13. NO INPUT VALIDATION: Population, generations, dimensions
    - User could request unreasonable values → crash
    - FIX: Add Pydantic field validators with bounds

14. NO LOGGING: Critical services (goldbeck_generator.py, optimizer.py, financial_calculator.py, shadow_calculator.py)
    - Debugging impossible
    - FIX: Add logging.getLogger() and logger.info() calls

15. INACCURATE GEOMETRY CONSTANTS: (geometry_engine.py:16-17)

    - FIX: Use 111,320m per degree + latitude-dependent lng

16. GENETIC ALGORITHM: Not adaptive (optimizer.py)

    - FIX: Implement progress-dependent mutation rates

17. WS STATE NOT PERSISTED: (ws_sync.py:43-47)

18. IFC MISSING DOOR OPENINGS: (ifc_exporter.py)
